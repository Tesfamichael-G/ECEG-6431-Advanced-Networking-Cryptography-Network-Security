Course Overview

This graduate‑level course covers socket programming, network management, number theory, public‑key cryptography, message authentication, digital signatures, network security protocols, wireless security, and system security. Students gain both theoretical foundations and practical skills.

Learning Objectives

  • ✅ Discuss concepts related to applied cryptography, including plaintext, ciphertext, symmetric/asymmetric cryptography, and digital signatures.
  • ✅ Understand the theory behind the security of different cryptographic algorithms.
  • ✅ Explain common network vulnerabilities and attacks, defence mechanisms, and cryptographic protection mechanisms.
  • ✅ Assess system security needs.
  • ✅ Discuss wireless network security.
